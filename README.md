# 🏠 Previsão de Preço de Casas com Machine Learning

Este projeto tem como objetivo prever o valor de casas com base em características como área, número de quartos e banheiros, utilizando Machine Learning com Python.

## 🚀 Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## 🧠 Sobre o projeto

Foi desenvolvido um modelo de regressão linear capaz de estimar o preço de imóveis com base em dados estruturados.

O projeto inclui:
- Leitura e tratamento de dados
- Treinamento de modelo de Machine Learning
- Interface interativa com Streamlit para entrada de dados
- Previsão em tempo real

## 📊 Variáveis utilizadas

- Área da casa (m²)
- Número de quartos
- Número de banheiros

## ⚙️ Como executar o projeto

1. Clone o repositório:
git clone https://github.com/seu-usuario/seu-repositorio.git

2. Instale as dependências:
pip install -r requirements.txt

3. Execute o projeto:
streamlit run app.py

## 💡 Exemplo de uso
O usuário informa:

Área da casa
Quantidade de quartos
Quantidade de banheiros

E o sistema retorna o valor estimado do imóvel.
